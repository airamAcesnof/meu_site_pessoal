# Diário de Desenvolvimento

## Passo 1

### Objetivo

Planear o website.

### O que fiz

- Defini o objetivo do site.
- Escolhi o estilo visual (portfólio moderno, dark e tecnológico).
- Desenhei o wireframe da estrutura.
- Defini as secções principais:
  - Hero
  - Sobre Mim
  - Competências
  - Projetos
  - Contacto

### Uma frase importante

Antes de escrever código é importante planear.

### Dificuldades

Nenhuma.

---

## Passo 2

### Objetivo

Criar a estrutura HTML do website.

### O que fiz

- Criei o ficheiro `index.html`.
- Estruturei o documento com HTML5 semântico.
- Adicionei as principais secções:
  - Header
  - Hero (apresentação com avatar + texto)
  - Sobre Mim
  - Competências
  - Projetos
  - Contacto
  - Footer
- Organizei o conteúdo de forma profissional e clara.
- Preparei o projeto para integração com CSS e JavaScript.

### Uma frase importante

O HTML define a estrutura do site, não o seu design.

### Dificuldades

Entender a organização correta das secções e a estrutura semântica.

---

## Passo 3

### Objetivo

Criar o design visual do website.

### O que fiz

- Criei o ficheiro `style.css`.
- Defini o tema visual (dark mode moderno).
- Estilizei o layout do Hero com avatar + texto lado a lado.
- Criei cards circulares para competências.
- Criei cards horizontais para projetos.
- Adicionei efeitos de hover e transições suaves.
- Estilizei contacto e footer.
- Melhorei a experiência visual geral do site.

### Uma frase importante

O CSS transforma estrutura em experiência visual.

### Dificuldades

Alinhar corretamente elementos com Flexbox.

---

## Passo 4

### Objetivo

Adicionar interatividade ao website.

### O que fiz

- Criei o ficheiro `script.js`.
- Adicionei scroll suave entre secções.
- Criei efeito de entrada ao carregar a página.
- Adicionei interações nos cards dos projetos.
- Testei o funcionamento no navegador.
- Adicionei mensagem de confirmação no console.

### Uma frase importante

O JavaScript dá vida ao website.

### Dificuldades

Nenhum
